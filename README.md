# m3u-editor

`m3u-editor` は、ローカルの音楽ファイルから M3U プレイリストを作成するデスクトップアプリです。

Tauri + React + Rust で実装し、ドラッグ＆ドロップで読み込んだ音楽ファイルを選択して、指定した出力先へシンプルな `.m3u` ファイルを書き出すことを目的にしています。

## 概要

このアプリは、音楽ファイルを管理しているフォルダやファイル群から、任意の曲だけを選んで M3U プレイリストを作成するためのシンプルなツールです。

初期実装では、プレイリスト編集に必要な最小機能に絞ります。音声再生やタグ編集のようなメディアプレイヤー機能は持たせず、M3U ファイルの作成に集中します。

## 主な機能

- ファイルまたはフォルダをドラッグ＆ドロップで読み込む
- フォルダを読み込んだ場合は、配下の音楽ファイルを再帰的に検出する
- 検出した音楽ファイルを一覧表示する
- 一覧から M3U に出力する音楽ファイルを選択する
- 表示されている順番で M3U に出力する
- M3U に書き出すパス形式を、絶対パス / 相対パスから選択する
- 保存ダイアログで出力先の `.m3u` ファイルを指定する

## 仕様

### 対象ファイル

音楽ファイルとして扱う拡張子は以下です。

- `mp3`
- `flac`
- `wav`
- `m4a`
- `aac`
- `ogg`
- `opus`
- `wma`

拡張子の判定は大文字小文字を区別しない想定です。

### 入力

- ユーザーはファイルまたはフォルダをアプリへドラッグ＆ドロップします。
- ファイルが渡された場合は、対象拡張子のファイルのみを一覧に追加します。
- フォルダが渡された場合は、フォルダ配下を再帰的に走査し、対象拡張子のファイルのみを一覧に追加します。

### 選択と並び順

- 検出した音楽ファイルは画面上に一覧表示します。
- ユーザーは一覧から M3U に含めるファイルを選択します。
- M3U への出力順は、画面に表示されている順番とします。

### 出力

- 出力形式はシンプルな M3U とし、ファイルパスのみを書き出します。
- `#EXTM3U` や `#EXTINF` は初期実装では出力しません。
- パス形式はアプリ上で以下から選択できるようにします。
  - 絶対パス
  - 出力する `.m3u` ファイルから見た相対パス
- 出力先は保存ダイアログで `.m3u` ファイル名まで指定します。

出力例:

```m3u
/home/user/Music/album/song01.mp3
/home/user/Music/album/song02.flac
/home/user/Music/album/song03.wav
```

## 初期スコープ外

初期実装では、以下の機能は対象外とします。

- `#EXTM3U` / `#EXTINF` を含む拡張 M3U の出力
- 音楽ファイルのメタデータ編集
- 音声再生
- プレイリストの高度な編集
- ファイルのタグ情報を使った自動ソート

## 技術構成

- Frontend: React + TypeScript + Vite
- Backend: Tauri 2 + Rust
- Package manager: pnpm
- Development environment: Nix flake + direnv

## 開発環境

このプロジェクトは Nix flakes と direnv を使って開発環境を構築します。

初回は以下を実行します。

```bash
direnv allow
pnpm install
```

`flake.nix` には、Rust stable、Node.js、pnpm、Tauri の Linux ビルドに必要な依存パッケージを定義しています。

## 開発コマンド

開発サーバーを起動します。

```bash
pnpm tauri dev
```

フロントエンドをビルドします。

```bash
pnpm build
```

Rust 側を確認します。

```bash
cargo check --manifest-path src-tauri/Cargo.toml
```
